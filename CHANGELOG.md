# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-18

### Added
- Initial release of CBC Disease Predictor
- AI-powered Anemia and Leukemia prediction
- Modern, responsive UI with glassmorphism design
- Animated components using Framer Motion
- Complete patient workflow from info to results
- Login and Signup pages (UI ready)
- Mobile-responsive navigation with hamburger menu
- Dynamic CBC value entry based on disease type
- Input verification page with data transformation view
- Color-coded prediction results
- Flask backend with ML model integration
- RandomForest model for Anemia detection
- XGBoost model for Leukemia detection
- Full responsive design (320px to 4K displays)

### Technical Features
- React 18 with Vite build tool
- TailwindCSS with custom animations
- Framer Motion for smooth transitions
- Flask API with CORS support
- scikit-learn and XGBoost ML models
- Formik and Yup for form validation
- React Router for client-side routing

### UI/UX
- Gradient color schemes throughout
- Floating background animations
- Hover and tap interactions
- Custom scrollbar styling
- Professional medical theme
- Accessibility features
